# **App Name**: Career Edge

## Core Features:

- User Authentication: Secure authentication for candidates, companies, and admins using distinct Firebase profiles.
- Role-Based Dashboards: Customizable dashboards providing role-specific functionalities and data views.
- Job Listings: Module for browsing, applying, and tracking job applications (candidate-side).
- Job Creation: Module for creating and managing job postings (company-side).
- Applicant Tracking System (ATS): System for companies to track and manage job applicants through the hiring process.
- AI-Powered Resume Analysis Tool: AI tool to scan and process a resume, classify key components and determine best matching role; accessible within ATS.
- Admin Account Management: Tools to manage all user accounts and app settings, with content moderation features.

## Style Guidelines:

- Primary color: Gradient from #667EEA to #764BA2 for a modern, AI-focused feel.
- Secondary color: Light teal (#A7FFEB) as a complementary color to the primary gradient.
- Background color: Very light gray (#F7FAFC) to provide a clean and modern backdrop.
- Accent color: Electric purple (#BC13FE) to add a pop of color and highlight interactive elements.  Consider this color for primary action buttons.
- Body font: 'Inter', a highly readable and modern sans-serif font for body text.
- Headline font: 'Poppins', a geometric sans-serif font for headings and titles.
- Use a modern and consistent icon set, such as 'Remix Icon' or 'Tabler Icons', with a focus on clean lines and minimalist design.
- Subtle animations using 'Framer Motion' for transitions and interactive elements to create a smooth and engaging user experience.  Focus on smooth transitions, and avoid excessive or distracting animations.
